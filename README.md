# 📊 MongoDB Assignment Project

## 📌 Overview
This project contains solutions to a set of MongoDB queries covering fundamental to advanced database concepts such as filtering, aggregation, joins, array operations, and nested queries.

All queries are implemented using MongoDB query language and aggregation framework.

---

## 🗂️ Dataset Collections Used
The project is based on the following collections:

- `students`
- `faculty`
- `courses`
- `enrollments`
- `activities`

---

## 🎯 Objectives
The main objectives of this assignment are:

- To understand MongoDB query operations
- To work with aggregation pipelines
- To perform joins using `$lookup`
- To manipulate arrays and nested documents
- To apply real-world data querying scenarios

---

## 🛠️ Technologies Used

- MongoDB
- MongoDB Compass / Shell
- JSON Data Format

---

## 📚 Query Categories

### 1. Complex Filters & Projections
- Filter students based on attendance and skills
- Identify faculty teaching more than a specified number of courses

### 2. Joins and Aggregations
- Use `$lookup` to combine collections
- Apply `$group` for counting and averaging data

### 3. Grouping, Sorting & Limiting
- Find top-performing students
- Analyze department-wise student distribution

### 4. Update, Upsert & Delete Operations
- Update attendance based on conditions
- Perform conditional upserts
- Delete outdated records

### 5. Array & Operator Usage
- Work with array fields like skills
- Apply operators like `$in`, `$nin`, `$all`

### 6. Subdocuments & Nested Conditions
- Query nested data with multiple conditions

### 7. Advanced Aggregation (Challenge Level)
- Multi-level joins and grouping
- Analytical queries like most popular activity types
